Recommended project file & folder structure

You can have whatever directory structure you like as long as you know how to the load modules. The following structure is recommended and we are using this structure though out our guide. 

project
|-app  
| |-modules
| | |-main
| | | |-index.html
| | 
| |-services ...
| |-index.html
|
|-node_modules
|-package.json

Galaxy.bootstrap

Add this code into the modules/main/index.html 

(function () {
  window.addEventListener('load', run);

  function run() {
    Galaxy.boot({
      // The path to your main module file
      url: 'modules/main/index.html'
    }, 
    function (module) {
      Galaxy.ui.setContent('body', module.scope.html);
    });
  }
})();

Hello world module

Create a directory inside modules/ directory and rename it to hello-world

Inside the hello-world directory create a html file called index.html and add the following code into it 

var state = Scope.import('state');

state.onStart = function (parameters) {
  console.log('Hello World!');
};
