Recommended project file & folder structure

You can have whatever directory structure you like as long as you know how to the load modules. The following structure is recommended and we are using this structure though out our guide. 

project
|-app  
| |-modules
| | |-main
| | | |-index.html
| | 
| |-services ...
| |-index.html
|
|-node_modules
|-package.json

Galaxy.bootstrap

Add this code into the app/index.html. 

<!doctype html>
<html>
  <head>
    <title>Learning GalaxyJS</title>
    <meta charset="UTF-8">

    <script src="galaxyjs/galaxy-min.js"></script>
    <script src="https://vuejs.org/js/vue.min.js"></script>    

    <script>
      (function () {
        // This will ensure that you boot Galaxy when everything is loaded
        // If you are using JQuery, you can also use $(document).ready(run);
        window.addEventListener('load', run);

        function run() {
          Galaxy.boot({
            // The path to your main module file
            url: 'modules/main/index.html'
          }, 
          function (module) {
            Galaxy.ui.setContent('body', module.scope.html);
          });
        }
      })();
    </script>
  </head>

  <body>
    Loading...
  </body>
</html>

Hello World!

Add the following code into the modules/main/index.html 

<h1>Hello World!</h1>

That's it, now you should be able to see Hello World! in your browser.

What happens here is that you bootstrap Galaxy and load your main module as soon as Galaxy done bootstrapping.
Then you add the content(html) of the main module to the body element by Galaxy.ui.setContent().

Lets add another module to our app.

Navigation module

Create a directory inside modules/ directory and rename it to navigation.

Inside the navigation directory create a html file called index.html and add the following code into it. 

<div ui-view="main"> 
  <a v-for="nav in navs" href="#{{ nav.id }}"> {{ nav.title }} </a>
</div>

<script>
  var vue = new Vue({
    el: Scope.views['main'],
    data: {
      navs: [
        {id: 'home', title: 'Home', path: 'modules/home/index.html'},
        {id: 'about', title: 'About', path: 'modules/about/index.html'}
      ]
    }
  })
</script>

We are going to load this module inside our main module.

You can also load navigation module in your bootstrap function, but in our guide we try to stick to best practices.

Add the following code into the modules/main/index.html 

<div ui-view="main-content"> </div>

<script>
  Galaxy.helpers.loadModuleInto({
    url: 'modules/navigation/index.html'
  }, Scope.views['main-content']);
</script>

Galaxy.helper.loadModuleInto() will load the module content into the specified element.

Now you should be able see your navigation module content on the page.
