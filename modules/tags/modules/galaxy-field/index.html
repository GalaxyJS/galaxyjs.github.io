galaxy-field Return to tags

Make boring input tags great again

galaxy-field reflects the state of its input as an attribute, therefore you can access them via CSS selectors. Everything you see here is achieved by CSS.

This text will go away when you fill in the text field Text field is filled

galaxy-field tries to provide a way to style different state of an input without using Javascript.

Here is the HTML and CSS for the above galaxy-field 

<galaxy-field>
  <label>This is a galaxy-field</label>
  <input title="This is a galaxy-field" type="text"/>
  <span class="empty">
    This text will go away when you fill in the text field
  </span>
  <span class="not-empty">
    Text field is filled
  </span>
</galaxy-field>

<style>
  galaxy-field {
    display: block;
    font-size: 15px;
    color: #5a5a5a;
    position: relative;
    padding-top: 38px;
    font-weight: 500;
    margin-bottom: 15px;
  }

  galaxy-field > label {
    display: block;
    font-weight: 500;
    padding: 13px 0 0;
    position: absolute;
    top: 0;
    left: 0;
    transition: all .3s ease-in-out;
  }

  galaxy-field > span {
    font-size: 13px;
    color: #888;
    display: block;
    line-height: 20px;
    height: 0;
    opacity: 0;
    overflow: hidden;
    padding: 0 2px;
    text-align: right;
    transition: all .3s ease-in-out;
  }

  galaxy-field > input, galaxy-field > select {
    font-weight: 500;
    font-size: 15px;
    line-height: 20px;
    background-color: #eee;
    width: 100%;
    display: block;
    padding: 10px 8px;
    border: 2px solid transparent;
    border-radius: 5px;
    margin-bottom: 5px;
    outline: none;
  }

  galaxy-field:not([empty]) > .not-empty {
    color: #009926;
    height: 20px;
    opacity: 1;
  }

  galaxy-field[empty] > .empty {
    height: 20px;
    opacity: 1;
  }

  galaxy-field[empty] > label {
    top: 38px;
    left: 10px;
  }

  galaxy-field[focus] > label {
    color: #009688;
    top: 0;
    left: 0;
  }

  .field[focus] > input, .field[focused] > select {
    border-color: #009688;
  }
</style>
